This repository contains a collection of end-to-end testing frameworks and projects demonstrating my proficiency in modern Software Testing Life Cycle (STLC) practices and automation.

Project Portfolio
1. API Testing Framework
Tools: REST-Assured / Postman / Newman
Description: A robust testing suite for validating RESTful APIs.
Key Features:
Validation of HTTP status codes, headers, and JSON payloads.
Automated data-driven testing using external data sources.
Integration with CI/CD pipelines to ensure backend stability.
2. BDD Framework (Cucumber)
Tools: Cucumber, Gherkin, Selenium Webdriver
Description: A Behavior-Driven Development (BDD) framework that bridges the gap between technical teams and stakeholders.
Key Features:
Wrote reusable Gherkin feature files (Given/When/Then).
Implemented step definitions for automated web browser interaction.
Generated clear, readable HTML test execution reports.
3. Page Object Model (POM) Design
Tools: Selenium, Java/C#, TestNG/JUnit
Description: A project demonstrating clean code principles through the Page Object Model design pattern.
Key Features:
Decoupled test scripts from UI elements to reduce code duplication and maintenance.
Enhanced script readability and reliability.
Implemented a central "Object Repository" for easy UI update management.
